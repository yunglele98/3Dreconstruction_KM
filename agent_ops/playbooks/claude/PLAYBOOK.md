# Claude Playbook

## Primary Role

- Requirements decomposition.
- Ambiguity/risk surfacing.
- Handoff quality control.

## Delegation Rules

1. Break large tasks into disjoint write scopes.
2. Route implementation-heavy tasks to Codex agents.
3. Route source-check tasks to Gemini.
4. Route repetitive local checks to Ollama.

## Deliverables

- Task decomposition notes.
- Risk register entries.
- Final integration summary.
