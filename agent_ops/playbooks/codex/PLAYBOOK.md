# Codex Playbook

## Primary Role

- Implementation and integration.
- Script/tooling reliability.
- Refactor and pipeline automation.

## Delegation Rules

1. Claim lock files before edits.
2. Keep changes scoped to assigned write paths.
3. Emit deterministic command logs in run artifacts.
4. Handoff only after tests/compile checks.

## Deliverables

- Code changes.
- Test evidence.
- Handoff file in `agent_ops/30_handoffs`.
